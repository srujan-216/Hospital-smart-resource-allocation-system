## Pronunciation fixes (if the AI mangles them)

If ElevenLabs mispronounces something, rewrite it phonetically:

- *Hungarian* → keep as-is, most voices handle it
- *sansne* → skip; this is in the form text, not the voiceover
- *I.C.U.* → write it with periods (as above) so the voice says I-C-U, not "ick-oo"
- *medico-legal* → if awkward, say *"hospital-grade record"*
